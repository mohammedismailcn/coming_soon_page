# Properdeal Coming Soon

Static coming-soon page for `properdeal.in` with an animated water background.

## Files

- `index.html` - main page
- `styles.css` - layout, typography, and water overlay styling
- `app.js` - Three.js animated water surface and ripple interaction
- `assets/properdeal-logo.png` - Properdeal logo

## Deploy

Upload the folder to Cloudflare Pages or GitHub Pages as a static site. No build command is required.

For Cloudflare Pages:

- Build command: leave empty
- Output directory: `.`
